# OneSiam Factory Web Catalog

แคตตาล็อกสินค้าแบบเว็บเดสก์ท็อป/มือถือ สำหรับงานฟาซาดและวัสดุอุตสาหกรรมของ `OneSiam Factory`

## ภาพรวม

โปรเจกต์นี้เป็นเว็บแอปแบบ Static ใช้ `HTML`, `CSS`, และ `Vanilla JavaScript` เพื่อแสดงสินค้า ค้นหา กรองหมวดหมู่ เปิดดูรายละเอียดสินค้า และรวบรวมรายการขอใบเสนอราคาในรถเข็นได้จากหน้าเดียว

## ฟีเจอร์หลัก

- แสดงสินค้าเป็นแคตตาล็อกแบบจัดหมวดหมู่
- ค้นหาสินค้าจากชื่อ รหัสสินค้า แท็ก และหมวดหมู่
- เรียงราคาจากต่ำไปสูง หรือสูงไปต่ำ
- เปิดหน้าแสดงรายละเอียดสินค้าแบบเต็ม
- แสดงสินค้าที่เกี่ยวข้องในหมวดเดียวกัน
- ตะกร้าขอใบเสนอราคา พร้อมสรุปราคารวม
- ฟอร์มข้อมูลลูกค้าเพื่อส่งคำขอใบเสนอราคา
- รองรับมือถือด้วยเมนูลัดด้านล่างและแผงค้นหาแบบย่อ
- ใส่โครงสร้าง SEO พื้นฐานผ่าน meta tags และ JSON-LD

## โครงสร้างไฟล์

```text

└── README.md
```

## การทำงานของระบบ

### `index.html`
- โครงสร้างหน้าเว็บไซต์หลัก
- ส่วน header, hero, ฟิลเตอร์หมวดหมู่, แคตตาล็อก, หน้ารายละเอียดสินค้า, รถเข็น และฟอร์มใบเสนอราคา
- เชื่อม `style.css` และ `app.js`

### `app.js`
- เก็บข้อมูลสินค้าใน `PRODUCTS_DATA`
- จัดการการค้นหา การกรอง การเรียง และการแสดงผลสินค้า
- สร้างหน้า detail view, related products, cart drawer และ checkout flow
- จัดการ localStorage สำหรับรถเข็น
- สร้าง JSON-LD product schema เพื่อช่วย SEO

### `style.css`
- กำหนด design system ของหน้าเว็บ
- คุม layout, สี, ระยะห่าง, component styles และ responsive behavior

## หมายเหตุสำคัญ

- สินค้าทดสอบ `comp-4` ถูกซ่อนจากหน้าร้านแล้ว
- สินค้าบางรายการใช้ราคาแบบ `ติดต่อสอบถาม`
- ปุ่มซื้อทำงานในแนวทางขอใบเสนอราคา ไม่ใช่ checkout ชำระเงินจริง

## วิธีใช้งาน

1. เปิด `index.html` ในเบราว์เซอร์
2. ใช้ช่องค้นหาหรือหมวดหมู่เพื่อหาสินค้า
3. คลิกการ์ดสินค้าเพื่อดูรายละเอียด
4. เพิ่มลงรถเข็นเพื่อขอใบเสนอราคา
5. กรอกข้อมูลติดต่อและส่งคำขอ

## การรันแบบ Local

ถ้าต้องการเปิดผ่าน local server สามารถใช้ตัวอย่างนี้ได้

```bash
python -m http.server 8000
```

แล้วเปิด `http://localhost:8000`

## ข้อมูลเพิ่มเติม

- รองรับการใช้งานบนเดสก์ท็อปและมือถือ
- ใช้ไอคอนจาก Font Awesome
- ใช้ฟอนต์จาก Google Fonts
- โครงสร้างหน้าและข้อมูลเหมาะสำหรับปรับต่อเป็นงานขายจริงหรือ landing page เชิงพาณิชย์

## สถานะโครงการ

อัปเดตล่าสุดตามการปรับปรุงแคตตาล็อก, หน้ารายละเอียด, รถเข็น, และเอกสารประกอบโครงการ
